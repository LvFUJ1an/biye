# IT技术社区分享平台

这是一个基于Vue 3 + Vite的IT技术社区分享平台，旨在为IT行业的开发者和爱好者提供一个交流、学习和分享的在线空间。

## 功能特点

- 用户注册/登录
- 博客发布与阅读
- 技术问答
- 视频资源分享
- 电子书资源
- 投票功能
- 实时通知
- 用户关注
- 搜索功能

## 项目设置

```bash
# 安装依赖
npm install

# 开发环境运行
npm run dev

# 打包构建
npm run build

# 预览构建结果
npm run preview
```

## 技术栈

- Vue 3
- Vue Router
- Pinia
- TypeScript
- Element Plus
- Tailwind CSS
- Vite

## 项目结构

```
src/
├── assets/         # 静态资源
├── components/     # 组件
│   ├── content/    # 内容相关组件
│   ├── layout/     # 布局组件
│   └── sidebar/    # 侧边栏组件
├── router/         # 路由配置
├── stores/         # Pinia状态管理
├── views/          # 页面视图
└── App.vue         # 根组件
```
