# IT技术交流与分享平台管理系统

这是一个基于Vue 3、Composition API、TypeScript的技术分享平台管理系统，该系统主要实现了管理员对用户、板块、文档、视频等内容的管理功能。

## 系统功能

- **用户管理**：管理员可以管理全站用户，可禁用/启用用户，查看用户详细数据
- **板块管理**：发布不同种类的板块，供用户在不同的板块下发布博客
- **数据统计**：通过仪表盘数据页面查看全站详细数据
- **文档中心**：管理员可上架和下架发布的电子书
- **视频中心**：管理员可管理用户在视频中心上传的视频资源

## 技术栈

- **前端框架**：Vue 3 + Composition API
- **类型系统**：TypeScript
- **构建工具**：Vite
- **路由管理**：Vue Router
- **HTTP请求**：Axios

## 开发与运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```
